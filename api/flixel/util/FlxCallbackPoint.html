class flixel.util.FlxCallbackPoint extends FlxPoint

import flixel.util.FlxPoint;

Available on all platforms

A FlxPoint that calls a function when set_x(), set_y() or set() is called. Used in FlxSpriteGroup. IMPORTANT: Calling set(x, y); is MUCH FASTER than setting x and y separately. Needs to be destroyed unlike simple FlxPoints!
