package flixel.addons.effects

FlxTrail

Nothing too fancy, just a handy little class to attach a trail effect to a FlxSprite. Inspired by the way "Buck" from the inofficial #flixel IRC channel creates a trail effect for the character in his game. Feel free to use this class and adjust it to your needs.

FlxTrailArea

This provides an area in which the added sprites have a trail effect. Usage: Create the FlxTrailArea and add it to the display. Then add all sprites that should have a trail effect via the add function.
