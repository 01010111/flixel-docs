static function generateCaveMatrix(Columns:Int, Rows:Int, ?SmoothingIterations:Int, ?WallRatio:Float):Array<Array<Int>>

Generate the matrix (2-dimensional array of Ints) for a cave.

@param Columns Number of columns for the matrix @param Rows Number of rows for the matrix @param SmoothingIterations How many times do you want to "smooth" the caev - the higher the smoother, but slower @param WallRatio Chance for a tile to become a wall - the closer the value is to 1.0, the more walls there are @return Returns a matrix of a cave!
