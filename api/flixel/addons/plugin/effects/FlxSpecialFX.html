class flixel.addons.plugin.effects.FlxSpecialFX extends FlxPlugin

Available on all platforms

FlxSpecialFX is a single point of access to all of the FX Plugins available in the Flixel Power Tools

Class Fields

static function clear():Void

Removes all FX Plugins. This is called automatically if the plugin is destroyed, but should be called manually by you if you change states.

static function isActive(Effect:BaseFX):Bool

Returns the active state of the given FX Plugin running

@param Effect A reference to the FX Plugin you wish to run. If null it will start all currently added FX Plugins @return Boolean true if the FX Plugin is active, false if not

static function remove(Effect:BaseFX):Bool

Removes a FX Plugin from the Special FX Handler

@param Effect The FX Plugin to remove @return Boolean true if the plugin was removed, otherwise false.

static function starfield():StarfieldFX

Creates a Glitch Effect

@return GlitchFX

static function startFX(?Effect:BaseFX):Void

Starts the given FX Plugin running

@param Effect A reference to the FX Plugin you wish to run. If null it will start all currently added FX Plugins

static function stopFX(?Effect:BaseFX):Void

Stops the given FX Plugin running

@param Effect A reference to the FX Plugin you wish to stop. If null it will stop all currently added FX Plugins
